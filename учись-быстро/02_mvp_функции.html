🚀 MVP-функции приложения "Учись Быстро"

Минимально жизнеспособный продукт (MVP) должен решать 1 основную задачу: помочь пользователю понять, как решается учебная задача, которую он отсканировал.

🧩 1. Сканирование задачи

Описание: Пользователь делает фото задачи (например, с тетради или учебника).

Функции:

  • Камера + галерея
  • Автообрезка области текста
  • Подсказка: "Фокусируйся на одном задании"

🔍 2. OCR (распознавание текста)

Инструмент: Google ML Kit или Tesseract OCR

Формат вывода:

x^2 + 5x + 6 = 0

Проблемы:

  • Плохое освещение
  • Рукописный текст (можно оставить на потом)
  • Нестандартное форматирование (таблицы, списки)

🧠 3. Объяснение с помощью AI

Цель: Простым языком объяснить, как решать задачу.

API: OpenAI GPT-4 (или Claude/Mistral)

Промпт-пример:

`Ты — репетитор по математике. Объясни пошагово, как решить следующую задачу: x^2 + 5x + 6 = 0 Объясни для ученика 9 класса.`

Результат:

  • Пошаговое решение (текст)
  • Итоговый ответ
  • Возможность переспросить ("а почему так?")

🌐 4. Локализация / Языки

Языки в MVP:

  • Английский (EN)
  • Испанский (ES)
  • Хинди (HI)
  • Португальский (PT-BR)
    Опционально: Автоопределение языка + перевод ответа

⭐ 5. Избранное / История

Для MVP:

  • Сохранить последний результат
  • Кнопка "Сохранить в избранное"
  • Мини-история последних 5 задач

🛑 6. Ограничения (Freemium)

Ограничения для бесплатных пользователей:

  • 3 задачи в день
  • Без сохранения истории (или ограниченно)
  • Показывать рекламу после каждой 2 задачи

Подписка:

  • Безлимит
  • Отключение рекламы
  • Приоритет в очереди AI

📦 Что НЕ входит в MVP (но можно добавить позже)

  • 📚 Тематические подборки (алгебра, геометрия и т.д.)
  • 🎓 Режим подготовки к экзамену
  • 📈 Аналитика прогресса (навыки, темы)
  • 🎤 Голосовой ввод
  • 📹 Видеообъяснения
  • 🤳 Поддержка рукописного ввода

✅ Критерии успешного MVP

  • Рабочее сканирование задачи
  • AI корректно объясняет минимум 80% кейсов
  • Пользователи сохраняют или делятся решением
  • 10–20% пользователей возвращаются повторно в течение недели
